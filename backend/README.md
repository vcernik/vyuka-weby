# Backend
Backendových technologií je mnoho (Node.js, PHP, ...), můžeme využít např. nám známý **Python**, konkrétně k tomu použijeme framework [Flask](https://flask.palletsprojects.com/en/stable/), který je pro **pochopení jak funguje backend** skvělý.

## Framework Flask

**Ukázková aplikace**:
- online: https://vyuka-weby.onrender.com/
- složka `app-flask-demo` 

**Tutoriál**: [Flask Tutorial - GeeksforGeeks](https://www.geeksforgeeks.org/python/flask-tutorial/)


## Hosting
Backend můžeme rozjet v rámci vývoje snadno na vlastním počítači, ale pokud chceme projekt zpřístupnit na internetu, potřebujeme webhosting, který umí potřebné technologie (v našem případě Python).

Pro zkoušení mohou stačit následující služby, které mají verzi zdarma:

- **Render**
  - https://render.com
  - lze poměrně snadno napojit na Github repozitář – každá změna se pak propíše automaticky do webu
  - první spuštění aplikace chvíli trvá...
- **Python Anywhere**
  - https://www.pythonanywhere.com
  - jednoduchý webhosting, soubory musíme ale vkládat ručně (ve free verzi není možná automatizace vkládání změn z repozitáře Githubu)
  - musí se měsíčně obnovovat
